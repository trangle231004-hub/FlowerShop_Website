# Flower Shop Website
A responsive front-end flower shop website built with **HTML, CSS, JavaScript, and Bootstrap 4**.

## ✨ Features
- Responsive layout for desktop, tablet, and mobile.
- Home page sections: Hero, Categories, Flash Sale, New Collection, Latest Products, Best Sellers, Testimonials, FAQ, Footer.
- Interactive UI with JavaScript:
  - Live Flash Sale timer
  - User dropdown menu
  - Login / Register / Search modals (close by button, overlay, and `Esc` key)
- Animation effects using Animate.css and WOW.js.

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 4.6
- Font Awesome
- Animate.css + WOW.js
